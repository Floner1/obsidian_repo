Default arguments:
used in parameters for function
def price(list_price, discount = 0.00, tax = 0.05)
the =0.00 and = 0.05 are examples of the default arguments
when u call function, if specific value for argument is not specified, it will use default, otherwise use specified value

if u use default, then it has to be after any no default, so u can't put discount or tax infront of list_price

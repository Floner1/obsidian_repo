type casting:
type() : finds the type of variable
all of the below converts the data type of variable into another data type
str()
int()
float()
bool(), can use to check if value entered, bc if variable in bool is "" then will return false, eg name = bool(name), if name == "" then will return false

Concatenate strings: use str += concatenate or str = str + variable
eg str = "hello", str += " 5123" would be "hello 5123"

math library:
math.ceil(): always round up
math.floor(): always round down
